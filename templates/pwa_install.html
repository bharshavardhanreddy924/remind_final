Install ReMind App

Follow the instructions below to install ReMind on your device for the best experience.

{% if is_android %}

Install on Android

1

Tap on the menu button (three dots) in Chrome

2

Select "Add to Home screen" or "Install app" from the menu

3

Confirm by tapping "Add" or "Install" in the popup

4

The ReMind app will now appear on your home screen!

Quick Install: You can also click the install button below to trigger the installation prompt directly.
{% endif %} {% if is_ios %}

Install on iPhone or iPad

1

Tap the share button at the bottom of Safari

2

Scroll down and tap "Add to Home Screen"

3

Tap "Add" in the top right corner

4

The ReMind app is now installed on your home screen!

Note: On iOS, you must use Safari to install the app. Chrome and other browsers do not support this feature on iOS.
{% endif %} {% if not is_android and not is_ios %}

Install on Desktop

1

Look for the install icon in the address bar (like a small computer with a down arrow)

2

Click the install icon and then click "Install" in the popup

3

The ReMind app will open in its own window and will be available in your start menu or desktop

Quick Install: You can also click the install button below to trigger the installation prompt directly.

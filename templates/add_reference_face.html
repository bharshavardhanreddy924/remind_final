About Reference Faces
What are reference faces?

Reference faces are stored images of people that the system uses to identify individuals in uploaded photos.

Tips for adding reference faces:
  • Use clear, well-lit photos showing the person's face
  • Front-facing photos work best for facial recognition
  • Add multiple photos of the same person for better accuracy
  • Faces should be clearly visible and not obscured
Privacy Information:

All face data is stored locally on your device and is not sent to external servers.

How recognition works:

The system uses facial recognition technology to:

  1. Detect faces in uploaded photos
  2. Extract unique facial features
  3. Compare these features with stored reference faces
  4. Identify matches based on similarity
